## How the App Works

When the program starts, it shows a menu with 6 options:

1. Create a file  
2. View all files  
3. Delete a file  
4. Read a file  
5. Edit a file  
6. Exit  

The user can choose any option by entering a number.  
Then the program performs that action and shows the menu again until the user selects "Exit".
